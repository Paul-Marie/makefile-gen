# makefile-gen
makefile-gen is a ruby script helping you by generating your C/C++/ASM(x86_64) Makefiles

### Installation

```sh
$ ./install.sh
```
### Execution
```sh
$ makefile-gen
```
getting usage
```sh
$ makefile-gen -h
```
